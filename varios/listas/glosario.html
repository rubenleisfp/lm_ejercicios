Glosario de Términos Informáticos

Algoritmo
Conjunto de instrucciones o reglas definidas que permiten llevar a cabo una actividad mediante pasos sucesivos que no generen dudas a quien deba hacer dicha actividad.
Bug
Defecto o fallo en un programa que impide que éste funcione de forma correcta.
Compilador
Programa que traduce el código fuente de un programa a un lenguaje de bajo nivel (código máquina) que puede ser ejecutado directamente por el ordenador.
Debugger
Programa utilizado para realizar la depuración de un programa, permitiendo la ejecución paso a paso y el análisis de las variables en tiempo de ejecución.
Framework
Conjunto estandarizado de conceptos, prácticas y criterios para enfrentar un tipo común de problema, que puede servir como referencia para enfrentar problemas similares.
